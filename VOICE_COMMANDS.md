## Tips for Best Results

### ‚úÖ DO:
- Speak clearly and at normal pace
- Wait for the "Listening" indicator
- Use your selected language (matches UI language)
- Speak in a quiet environment
- Hold device close enough to microphone

### ‚ùå DON'T:
- Speak too fast or too slow
- Speak when "Listening" indicator is off
- Mix multiple languages in one command
- Use in very noisy environments
- Expect instant results (allow 1-2 seconds)

## Troubleshooting

### Problem: "Voice features not supported"
**Fix:** Use Chrome, Edge, or Safari

### Problem: Microphone not working
**Fix:** 
1. Check browser permissions (lock icon in address bar)
2. Allow microphone access
3. Refresh page

### Problem: Commands not recognized
**Fix:**
1. Speak more clearly
2. Check language selector matches your speech
3. Try English first to test if it's working
4. Reduce background noise

### Problem: No voices for text-to-speech
**Fix:**
1. Wait 5-10 seconds after page load
2. Check browser language settings
3. On mobile: download language pack

## Privacy

- All processing happens in your browser
- No recordings are stored
- Transcript clears when you stop listening
- You can disable anytime via Settings
